# 素数个数

# Description

给定两个非负整数 a，b，其中 0<= a,b<=1,000,000，请计算这两个数之间有多少个素数。
输入
第一行是一个整数 K（1<=K<=1000），表示有多少个样例，每个样例占一行，是两个整数 a 和 b，每个整数之间用一个空格隔开。
输出
每行输出一个样例的结果。
