# KALI LINUX 利用脚本实现 DDOS-ATTACK

# 环境

kali-everything 2022 镜像文件
VMware Workstation Pro

# 步骤

  • 下载配置文件（HA-MRX 大佬的脚本）（目前在用）
    其他脚本在此暂不提及。
  • 进入 root 模式

    输入账号密码，密码默认不显示。
  • 前往 DDos-Attack 文件夹
  • 修改（设置）对 ddos-attack.py 文件执行的权限
  • 执行 ddos-attack.py 程序
  • 执行攻击

    如若成功此处会提示输入 IP 地址。
    打开一个命令行窗口，ping 一下目标网址获得 IP 地址
    点击回车
    Port 输入线程数（推荐 80）
    原因：此次攻击并没有利用肉鸡（貌似），线程数值越高，耗能也越大，效果也会越好，但线程数如果超过电脑可承受范围，会导致进程卡死。
    回车运行，按下 ctrl C 退出。

# 已知报错
