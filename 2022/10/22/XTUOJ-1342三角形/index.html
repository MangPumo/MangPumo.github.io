# 三角形

# 题目描述

已知两个三角形的三条边的长度，请问它们是否相似？

# 输入

第一行是一个整数 T (1≤T≤1000)，表示样例的个数。 每个样例占两行，每行为三个整数 a,b,c1≤a,b,c≤1000, 表示一个三角形的边长。 输入数据保证三条边一定能构成三角形。

# 输出

依次输出每个样例的结果，如果相似，输出 "Yes"，否则输出 "No"。
